喜歡佐佐木還有涼宮春日系列的朋友们大家好！文艺部的各位好久不見！這里是 Yoccoyouto。

『佐佐木的虚樣日常』是這邊的第一部日文譯作，同樣也是作为研究學習日語的一環。在本土化与異域風情之间，筆者會努力做出平衡的。

『佐佐木的虚樣日常』的作者飛鳥彼方先生在大約十年前創作了這部長篇同人作品，并且直到新冠大流行期间一直保持更新狀態。

翻譯的過程是先由大模型給出一個可能的版本，再用鍵盤比照原文進行調整和润色。正如標題所示，本作（截至目前所譯部分）着重描繪的是日常生活元素；如果能以中文呈現出作者筆下的細膩情感，如果大家能够舒服地阅讀下去，我會很高興。

想问问讀者朋友们幾個问題，不過暫時還沒想得很好，畢竟也是邊讀邊譯（因此按理說後面會有一個統籌修正環節，但不一定有空）。請问：
1. 最後會有幾對情侶呢？
2. 在一節裡面莫名其妙的闪回還是獨白云云，大家會很在意嗎？多一点好，或者少一点好？

我有点感興趣。

在翻譯的過程中也會修正作者可能的筆誤；如果譯文有所錯漏也是人之常情，看看是我犯的錯多還是語言模型和翻譯姬的多？說笑了。歡迎大家指正疏漏之處和提意見，這邊會侭快做出回應的。
